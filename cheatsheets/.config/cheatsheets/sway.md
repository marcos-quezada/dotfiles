# Sway Keybindings

> Mod = Super (Windows key)

## Window Management
| Key                    | Action                                  |
|------------------------|-----------------------------------------|
| `Mod+Enter`            | Open terminal                           |
| `Mod+d`                | Open launcher (wofi / dmenu)            |
| `Mod+Shift+q`          | Kill focused window                     |
| `Mod+f`                | Toggle fullscreen                       |
| `Mod+Shift+Space`      | Toggle floating                         |
| `Mod+Space`            | Switch focus: tiling ↔ floating         |
| `Mod+h/j/k/l`          | Move focus left/down/up/right           |
| `Mod+Shift+h/j/k/l`   | Move window left/down/up/right          |
| `Mod+r`                | Enter resize mode (then h/j/k/l)        |

## Layout
| Key            | Action                                  |
|----------------|-----------------------------------------|
| `Mod+e`        | Toggle split direction                  |
| `Mod+s`        | Stacked layout                          |
| `Mod+w`        | Tabbed layout                           |
| `Mod+b`        | Split horizontal                        |
| `Mod+v`        | Split vertical                          |

## Workspaces
| Key                  | Action                                  |
|----------------------|-----------------------------------------|
| `Mod+1..0`           | Switch to workspace 1–10               |
| `Mod+Shift+1..0`     | Move window to workspace 1–10          |

## System
| Key                  | Action                                  |
|----------------------|-----------------------------------------|
| `Mod+Shift+c`        | Reload sway config                      |
| `Mod+Shift+e`        | Exit sway                               |
| `Mod+Shift+r`        | Restart sway in-place                   |

## Custom
| Key            | Action                                  |
|----------------|-----------------------------------------|
| `Mod+?`        | Open this cheatsheet popup              |
